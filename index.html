Trabajo Práctico 4 - Tablas

Catalina Petasny

1. ¿Cuál es la etiqueta principal utilizada para crear una tabla en HTML?

R: La etiqueta principal que se usa para crear una tabla en HTML es <table>, y se cierra con </table>


2. ¿Qué etiqueta se utiliza para definir una fila en una tabla?

R: La etiqueta que se usa para una fila es <tr>


3. ¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

R: La etiqueta que se utiliza para crear una celda de encabezado en una tabla es la etiqueta <th> y se cierra con </th>


4. ¿Cuál es la etiqueta utilizada para crear una celda de datos en una tabla?

R: La etiqueta que se utiliza para crear una celda de datos en una tabla es el atributo colspan o rowspan.
A la etiqueta <th> se le aplica el atributo colspan. Este atributo hace que esa fila ocupe dos columnas. Así mismo
el atributo rowspan funciona de la misma manera pero hace que la celda se extienda hacia abajo en varias filas.


5. ¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

R: El atributo que se usa para especificar cuantas columnas debe abarcar una celda es colspan, este mismo
se le aplica al <th> y al momento de agregar el colspan se le agrega colspan="x" lo que significa cuantas
columnas se le agrega a la celda.


6. ¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

R: El atributo que se utiliza para especificar cuántas filas debe abarcar una celda es el "rowspan". Este funciona de la
misma manera que el colspan. Hay que escribir rowspan="x" y el número de filas que se quiere agregar. Este atributo hace
que la celda se extienda hacia abajo en varias filas.


7. Si necesitamos crear una tabla con 10 filas ¿qué etiqueta y cuántas veces se debe utilizar para definir las filas?

R: Para crear una tabla con 10 filas, la etiqueta que se debe utilizar es <th> y para definirla se debe usar 10 veces.
